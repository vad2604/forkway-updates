Forkway

Menu bar-приложение для split-маршрутизации: CheckPoint SNX, AmnesiaWG, sing-box.

macOS 14+, Apple Silicon (arm64).

Скачать DMG (1.0.4) SHA256

Установка

  1. Откройте DMG → перетащите Forkway в Applications.
  2. В Terminal: 
    xattr -cr /Applications/Forkway.app
/Applications/Forkway.app/Contents/MacOS/ForkwayApp
  3. Иконка появится в menu bar (не в Dock).

Amnezia VPN

  1. В Amnezia VPN: Настройки → Создать резервную копию → сохраните .backup.
  2. Отключите VPN и закройте Amnezia перед запуском Forkway.
Важно: пока Forkway работает (Start), Amnezia должна быть выключена. Два VPN-клиента одновременно конфликтуют за маршруты.

Первый запуск

  1. Forkway → Import AWG from Amnezia… → выберите .backup
  2. Amnezia отключена
  3. Подключите CheckPoint
  4. Start (пароль admin — один раз)

Обновления

Автоматически через Sparkle. Вручную: меню Forkway → Check for Updates…

Проверка DMG

cd ~/Downloads
